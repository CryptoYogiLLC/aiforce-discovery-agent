=============================================================================
GETTING STARTED
=============================================================================

1. Get the Approval UI URL:
{{- if .Values.approvalUi.ingress.enabled }}
{{- range $host := .Values.approvalUi.ingress.hosts }}
  http{{ if $.Values.approvalUi.ingress.tls }}s{{ end }}://{{ $host.host }}
{{- end }}
{{- else }}
  kubectl port-forward --namespace {{ .Release.Namespace }} svc/{{ include "discovery-agent.fullname" . }}-approval-ui 3000:{{ .Values.approvalUi.service.port }}
  Then open: http://localhost:3000
{{- end }}

2. Check the status of all pods:
  kubectl get pods --namespace {{ .Release.Namespace }} -l "app.kubernetes.io/instance={{ .Release.Name }}"

3. View logs for a service:
  kubectl logs --namespace {{ .Release.Namespace }} -l "app.kubernetes.io/instance={{ .Release.Name }},app.kubernetes.io/component=network-scanner"
